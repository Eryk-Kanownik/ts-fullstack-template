## Typescript Fullstack Template

### Stack

1. **Green** Node.js
2. **Blue** Typescript (Both frontend and backend)
3. **Gray** Database is up to you
4. **LightBlue** React
5. **Purple** Redux
6. **Pink** SCSS

### Commands to run

1. npm run install-backend
2. npm run install-frontend
3. npm run dev-app
